SSHFling 0.1.11 community package manifests

SSHFling is proprietary commercial software. Installing, running, redistributing, or submitting these manifests to third-party repositories requires the rights described in the project LICENSE or a separate written agreement from GRWLX.

These files are generated from the release artifacts. Some ecosystems can install directly from these URLs; official/community repositories such as AUR, FreeBSD Ports, pkgsrc, winget, Chocolatey, Snapcraft, and distro repos still require maintainer account submission and review.

  • Arch / AUR: PKGBUILD, .SRCINFO
  • Alpine: APKBUILD
  • FreeBSD Ports: security/sshfling port
  • OpenBSD Ports: security/sshfling port
  • pkgsrc for NetBSD, DragonFly BSD, illumos, and SmartOS: security/sshfling package
  • Nix: flake.nix
  • Guix: sshfling.scm
  • Void Linux: xbps-src template
  • Gentoo: ebuild
  • Slackware: SlackBuild, slack-desc
  • openSUSE OBS: spec file
  • Snapcraft: snapcraft.yaml
  • Termux: package build.sh
  • AppImage: AppImageBuilder.yml
  • Scoop: manifest
  • winget: multi-file manifest directory
  • Chocolatey: nupkg, nuspec

Fast Commands

curl -fsSLO https://grwlx.github.io/sshfling/arch/PKGBUILD && makepkg -si
scoop install https://grwlx.github.io/sshfling/scoop/sshfling.json
irm https://grwlx.github.io/sshfling/chocolatey/install.ps1 | iex
